## Capture guidance

- Keep the same bracelet project visible across multiple screenshots where possible. The page will feel more coherent.
- Use one visually strong pattern with enough color variation to show off palette handling.
- Avoid blank canvases or tiny maps. The design should feel substantial.
- Prefer full-device screenshots without annotations. I can frame and crop them in the landing page later.
- Keep system status bars and window chrome visible unless they look distracting.
- For macOS, avoid cluttered desktops. The app window should dominate the frame.
- For iPhone screenshots, avoid modal stacks unless the modal itself is the feature being shown.

## Suggested shot plan

- iPhone hero main: editing canvas
- iPhone hero secondary: completed beads or palette
- iPhone editing card: brush or erase action
- iPhone palette card: palette toolbar or replace color
- iPhone progress card: counts or marked beads
- Export card: printable PDF
- macOS hero: full workspace
- macOS detail: alternate workspace angle
- iPad extra: larger touch layout
